This folder is intended for raw RNA-seq FASTQ files (paired-end).

Example files:
- SRR5924196_1.fastq.gz
- SRR5924196_2.fastq.gz
- SRR5924198_1.fastq.gz
- SRR5924198_2.fastq.gz

Due to size constraints, FASTQ files are not stored in this repository.
They can be downloaded from public repositories (e.g., ENA/SRA).
